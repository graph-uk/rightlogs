Что такое логи?

  • Reflects inner state & events
  • Computer-generated
  • Human readable / Plain text
  • Contains date

ELK

  • LogStash. Multi-purpose log processing
  • Elastic search. No-sql database which speaks Rest.
  • Kibana. Visualization platform. Pure JavaScript. Latest version requires node.js.

ELK Extended

  • NLog / Log4Net. Logging framework for applications
  • LogCourier. Multi-line CRLF processing + text log monitoring to send to LogStash.
  • Riemann. Event aggregator / Notification platform.

Demo: Configuring NLog

Log Levels

  • Fatal. Lead to close / Harm users
  • Error. App can still run / One user affected
  • Warn. Potentially harmful situation.
  • Info. Something happened in case you want to handle this.
  • Trace/Debug. :)

Log Courier + Log Stash + Kibana demo

Logging Best practices

  • Log rotation is your friend
  • Log RequestID across same web request
  • Log with context.
    USER=wp34056 - action=login - page=blog.logentries.com
  • Don't log sensitive information :)
